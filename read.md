# Reyman D. Casio — Portfolio Website

A beautifully redesigned glassmorphism portfolio with smooth interactive animations.

## File Structure

```
portfolio/
├── index.html     — Main HTML structure & content
├── style.css      — All styles (glass UI, animations, responsive)
├── app.js         — All JavaScript (cursor, animations, interactions)
└── README.md      — This file
```

## Features

- **Glassmorphism design** — Deep space theme with frosted glass cards
- **Custom cursor** — Animated cursor with trail effect (desktop)
- **Interactive hover** — Card tilt (3D), ripple clicks, particle effects
- **Animated background** — Floating orbs that react to mouse movement
- **Scroll reveals** — Staggered entrance animations
- **Counter animations** — Stats count up on scroll
- **Tab switcher** — Skills organized by category
- **Project modals** — Detailed view with workflow images
- **Responsive** — Works on desktop, tablet, and mobile
- **Smooth performance** — GPU-accelerated transitions, will-change hints

## Deployment

Upload all 4 files to the same folder/directory. No build step required.
Works directly in any modern browser (Chrome, Firefox, Safari, Edge).
